Twobit's Command List

These are all commands that power Twobit and your server!

Moderation
Command Description Example
ban Ban any user from your guild. >ban [user] [reason]
kick Kick any user from your guild. >kick [user] [reason]
mute Mute any user from your guild. >mute [user] [reason]
warn Warn any user from your guild. >warn [user] [reason]
purge Delete previous messages instantly >purge [Number of messages]
Tickets
Command Description Example
create Create a ticket if you need assistance >ticket create
close Close your opened ticket >ticket close
rename Rename your ticket channel >ticket rename
Utils
Command Description Example
avatar View a person's discord avatar >>avatar or >avatar [user]
botstats A usefull command that displays bot statistics. >botstats
coinflip Flip a coin >coinflip
cr View the creator of the bot >cr
cr View the creator of the bot >cr
emoji Add a new emoji to your server within a few seconds >emoji [name] [url]
infractions View all infractions of a user [Warns, bans, kicks, mutes] >infractions [user]
invite Generate Twobit's invite link >invite
poll Create an instant poll >poll [poll title]
lock Lock a channel. No more messages >lock
unlock Unlock an already locked channel >unlockk
rate Fun command to rate something on a scale of 1 to 10 >rate
whois Get all the information of a user >whois or >whois [user]
Tags
Command Description Example
create Create a tag within a matter of seconds >tag create [tag name] [content]
info View a tag's info >tag info [tag name]
remove Remove an already existing tag >tag remove [tag name]
update Update an already existing tag >tag update [tag name] [new content]
